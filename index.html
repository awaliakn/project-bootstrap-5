Mau Belajar Apa Hari Ini?

HTML

HTML (Hypertext Markup Language) adalah bahasa markup untuk membuat halaman web menggunakan simbol tertentu (tag) yang nantinya akan diterjemahkan oleh browser ke halaman web. Dengan menggunakan tag-tag HTML.

Selengkapnya
CSS

CSS adalah singkatan dari cascading style sheet, yaitu bahasa yang digunakan untuk menentukan tampilan dan format halaman website. Dengan CSS, Anda bisa mengatur jenis font, warna tulisan, dan latar belakang halaman.

Selengkapnya
JavaScript

JavaScript adalah bahasa pemrograman populer yang digunakan untuk membuat situs dengan konten website yang dinamis artinya konten dapat bergerak atau berubah di depan layar tanpa perlu mereload halaman.

Selengkapnya
Bootstrap

Bootstrap adalah framework HTML, CSS, dan JavaScript yang berfungsi untuk mendesain website responsive dengan cepat dan mudah. Framework ini tersusun dari kumpulan file CSS dan JavaScript berbentuk class yang tinggal pakai.

Selengkapnya
PHP

PHP "PHP: Hypertext Preprocessor," adalah bahasa pemrograman yang digunakan secara luas untuk pengembangan aplikasi web dan server-side scripting. PHP menjalankan instruksi yang berbeda tergantung pada data yang diproses.

Selengkapnya
LARAVEL

Laravel adalah framework berbasis bahasa pemrograman PHP yang bisa digunakan untuk membantu proses pengembangan sebuah website agar lebih maksimal. Dengan menggunakan Laravel, website yang dihasilkan akan lebih dinamis.

Selengkapnya
